To work on this lab on your own machine, you will need to install POSTMAN, Python and the Python Requests library.

Install POSTMAN

  • Download POSTMAN and install.

    • Install Python

    • Download Python version 3.x and install.

    Install the Requests Library

    • You can read more about the Requests Library here
    • To install Requests for Python 3 on Mac OS: 
      •                 pip3 install requests
    • To install Requests on Windows:
      • Pip comes with Python 3.4 Windows
        •                     pip install requests
